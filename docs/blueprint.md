# **App Name**: SaudiDropship

## Core Features:

- Product Gallery: Display products in a responsive grid layout, with filtering by category.
- Product Details: Showcase individual products with an image, price, and description. Ensure all details from the user request are displayed.
- Description Copy: Allow users to copy the product description to their clipboard, with visual feedback on success.
- WhatsApp Order: Integrate a WhatsApp button that initiates a chat with a pre-filled message containing the product details for easy ordering.
- Product Management: Enable admin users to manage products (CRUD operations) via the Firebase console. Incorporate image upload functionality.
- Dynamic Content Generation: The app features an AI tool that can generate descriptions of the products by analyzing their main characteristics.
- Firestore Integration: Utilize Firestore to store and retrieve product data, including product names, prices, descriptions, and images.

## Style Guidelines:

- Primary color: Desert Sand (#D4A373) to reflect the natural landscape of Saudi Arabia.
- Background color: Off-White (#F8F5F1), a slightly desaturated version of the primary color to maintain visual connection but reduce intensity. Provides a neutral backdrop.
- Accent color: Deep Red (#A32719), an analogous color offering contrast and highlighting calls to action and important details. Influenced by the color of the Saudi Arabian flag.
- Headline font: 'Belleza' (sans-serif) for headlines and shorter descriptions. Body font: 'Alegreya' (serif) for product descriptions, or any potentially longer text.
- Use clear, simple icons for categories and actions, ensuring they are culturally appropriate for the KSA market.
- Mobile-first, responsive grid layout optimized for displaying products effectively on various screen sizes.
- Incorporate subtle loading animations and transitions to improve user experience during data fetching and interactions.
